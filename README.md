# Penguin Classification with Support Vector Machine

This project uses machine learning (Support Vector Machines) to classify Adelie penguins vs. non-Adelie penguins using two physical measurements:

- Flipper Length (mm)
- Bill Length (mm)

The program loads a dataset, cleans it, visualizes the data, trains a SVM classifier, and plot a decision hyperplane boundary.

# Requirements

Python version:

- Python 3.10 or 3.11

Libraries used:

- numpy (version 1.26.4)
- pandas (version 2.1.2)
- seaborn (version 0.13.2)
- matplotlib (version 3.8.0)
- scikit-learn (version 1.7.2)
